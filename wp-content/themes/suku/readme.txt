== Description ==

Suku is a free and minimal WordPress FSE theme. It is a Gutenberg-ready WordPress theme and is compatible with all blocks and patterns of WordPress. If you are building a portfolio or blog theme using FSE feature, you can use this theme. If you want to know what full site editing is, you can learn from here https://sitenerdy.com/wordpress-full-site-editing/
